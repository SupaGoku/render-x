# Product Mission (Lite)

Render-X is a micro JSX rendering toolkit that helps frontend engineers ship interactive experiences without heavyweight frameworks by providing React-compatible hooks, portals, and DOM bindings in sub-5KB packages.

Render-X gives performance-focused web teams a tiny yet expressive renderer with TypeScript-first ergonomics and native portal support, so they can migrate components from React without sacrificing hooks or hydration. Its dual-package architecture keeps bundles lean while delivering the debugging and tooling surface modern teams need to collaborate.
